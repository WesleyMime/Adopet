Cadastre um novo pet para adoção:

Digite um nome.
Nome deve ter pelo menos 3 caracteres.
Digite uma idade.
Idade deve ter pelo menos 4 caracteres.
Digite um endereço.
Endereço deve ter pelo menos 4 caracteres.
Digite o tamanho.
Tamanho deve ter pelo menos 8 caracteres.
Digite a descrição.
Descrição deve ter pelo menos 10 caracteres.
Digite a URL da imagem.
Imagem deve ser uma URL válida.
